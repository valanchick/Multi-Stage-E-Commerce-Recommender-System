# Multi-Stage-E-Commerce-Recommender-System
Цель системы — за миллисекунды отбирать релевантные товары из огромного каталога, максимизируя вероятность покупки (Conversion Rate), при этом сохраняя категориальное разнообразие (Diversity) в итоговой выдаче, чтобы избежать эффекта "каннибализации" витрины.
